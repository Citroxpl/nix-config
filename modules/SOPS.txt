SOPS setup new device (NixOS)

Generate age key (on NEW device):
   mkdir -p ~/.config/sops/age
   age-keygen -o ~/.config/sops/age/keys.txt

Copy public key:
   age-keygen -y ~/.config/sops/age/keys.txt

Add public key to .sops.yaml (on existing machine):
   creation_rules:
     - path_regex: secrets\.yaml$
       key_groups:
         - age:
             - <old-device-key>
             - <new-device-key>

IMPORTANT: update encrypted file (on existing machine):
   sops updatekeys secrets.yaml

Commit changes:
   git add .sops.yaml secrets.yaml
   git commit -m "Add new device to sops"

On new device:
   git clone <repo>
   cd nixosConfig

Build system:
   sudo nixos-rebuild switch --flake .#<hostname>
